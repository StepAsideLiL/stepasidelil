## Intruduction

Web is a complex ecosystem.

But it was not started this way. The first ever [website](https://info.cern.ch/) is just a html content served by a daemon called [httpd](https://www.w3.org/Daemon/).

This is the simplest form of website. A server that sends some markup text upon request.
