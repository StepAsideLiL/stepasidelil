# create-little-proj

Create a new little-proj project.

### Why create-little-proj?

create-little-proj is a node CLI tool to start a new little-proj project. It is the quickest way to create and modify files for the new project.

### Getting Started

If you are using pnpm, run the following command:

```bash
pnpm dlx @stepasidelil/create-little-proj
```

Answer the prompts and the project will be created.
